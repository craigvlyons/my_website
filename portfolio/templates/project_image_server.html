This project was designed to be a web site for my local media server at home. The current directory is displayed in the top corner. Images can be uploaded to the current file that you choose. All images in the selected file will be rendered to the page.

Problem that it solves.

Provides a quicker and easier way to upload images, than locating and pluging in a hard drive.

Built with HTML / CSS layout, styling of the web page. Python library Dropzone handles drag and drop image uploads. Python OS to travese files. Flask, Jinja2 for the request and user inputs.
